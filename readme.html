AdSaDi - Advanced SAP Dialog based on CL_GUI_HTML_VIEWER

AdSaDi - Advanced SAP Dialog is a smal ABAP package to make the SAP HTML Viewer (CL_GUI_HTML_VIEWER) ready to use along with dynamically created HTML content an runtime by using XSL transformations.
Here is a simple "Hello, World!" dialog:

1. Create an XSL stylesheet ZADSADI_HELLO_WORLD.XSL

     1 <xsl:transform version="1.0"
     2   xmlns:xsl="http://www.w3.org/1999/XSL/Transform"
     3   xmlns:sap="http://www.sap.com/sapxsl"
     4 >
     5 <xsl:template match="/">
     6 <html>
     7   <body>
     8     <div>Hello, <b>World</b>!</div>
     9   </body>
    10 </html>
    11 </xsl:template>
    12 
    13 </xsl:transform>

2. Create a program

     1 data: dlg type ref to /ukw/adsadi_dialog.
     2 
     3 create object dlg
     4   exporting
     5     i_xslt = 'ZADSADI_HELLO_WORLD.XSL'.
     6 
     7 if dlg->render( ) ne /ukw/adsadi_dialog=>dlg_cancel.
     8   write: / `success...`.
     9 else.
    10   write: / `cancel...`.
    11 endif.
